Aim:

To control the motion of a Servo motor using Arduino.

Simulation Environment:

TinkerCAD (Free online simulator)

Components:

Arduino UNO
Servo Motor

Theory:

A micro servo motor is a small-sized servo motor designed for applications where space is limited. Servo motors are devices that incorporate a feedback mechanism to control the speed and position of the motor accurately. They are commonly used in robotics, remote-controlled vehicles, and other projects where precise control of movement is required.

Motor: The motor inside the servo is responsible for producing mechanical motion. It typically consists of a DC motor.

Gear Train: Servos have a gear train that converts the high-speed, low-torque output of the motor into low-speed, high-torque motion.

Control Circuitry: The control circuitry interprets signals received from an external source (like Arduino) and translates them into precise movements.

Potentiometer (Feedback Device): Most servo motors contain a potentiometer that provides feedback to the control circuit about the current position of the motor.

When we connect a micro servo motor to an Arduino, we typically use the Servo library in Arduino to control its movements.

The movement of a servo motor attached to an Arduino is controlled by sending a series of pulses to the servo motor. These pulses are generated using a technique called Pulse Width Modulation (PWM).

Working:

  • Pulse Width Modulation (PWM): Arduino boards generate PWM signals where the pulse width determines the servo’s position.
  • Servo Library: The Arduino Servo library simplifies controlling servo motors by generating the required PWM signals.
  • Attach Function: The attach() function connects the servo object to a specific Arduino pin.
  • Write Function: The write() function sets the servo angle (0° to 180°).
  • Pulse Generation: The library converts the angle into an appropriate PWM signal and sends it to the servo.
  • Control Loop: The servo continuously adjusts its position based on feedback from the internal potentiometer.
  • Looping Control: Using a loop, different angles can be sent to the servo to create continuous or patterned movement.

Conclusion:

The servo motor was successfully controlled using Arduino by generating PWM signals. The position of the servo can be precisely adjusted using programming and the Servo library.

Arduino Code: Servo Motor Control


// Include the Servo library
#include <Servo.h>

Servo servoBase;   // Create a Servo object

void setup() {

  // Attach the servo to pin A1
  servoBase.attach(A1);

  // Set the servo motor to 0-degree position
  servoBase.write(0);
}

void loop() {

  // Sweep the servo from 0 to 180 degrees
  for (int i = 0; i <= 180; i += 10) {

    servoBase.write(i);   // Set servo to current angle
    delay(2000);          // Pause for 2 seconds
  }
}
